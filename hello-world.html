Hello World

Date Wed 05 August 2020 Tags Py3.8 / Python / Function

Welcome to a better, "Hello World". Let's start with the basics.

# This is a comment!
print("Hello World")

# Outcome
Hello World

Not bad, let's mess with that string a bit.

# What about using a separator, and using multiple strings?
print("Hello", "World", sep=' ')

# Outcome
Hello World

Same outcome! Let's specify the end parameter.

# The end parameter takes a string and simply places it at the end of the print function.
print("Hello", 'World', sep='-', end='!!')  # Let's use two bangs!!

# Outcome
Hello-World!!

So, about now you are thinking, "Wow, there are so many awesome ways to use the print function"! But now I want to blow your mind and introduce you to the wonderfully modular world of Python. There is a plugin called Colorama, which has the ability of making our output even more mind-blowing!!

Let's install it using pip.

# This command needs to be run in a terminal session, and not a python IDLE window.
pip install colorama

Installed Module's (or Library's) can be a bit more complicated to use then python's built-in ones. So always reference the documentation if you are confused. StackOverflow is also a great resource.

Let's initialize colorama, and use it! I'm not going to post the mind-blowing output and you will just have to do it yourself!

# First import colorama's init function.
from colorama import init

# Then initialize colorama
init()

# Then import the functions that will allow us to edit the Foreground, Background, and Style
from colorama import Fore, Back, Style

# Now let's make our "Hello World" Red Text, Black Background, and Dim Style.
# All of which represents world domination.
print(Fore.RED + Back.BLACK + Style.DIM)
# I can separate these because they are actually opening ASCII commands to recolor/stylize the font.
print("Hello World!!)
# We can set all the recolor's back to normal by running the following.
print(Style.RESET_ALL)

You are now a print function king. Go now, and conquer the world, one hello at a time!
